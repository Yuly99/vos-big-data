# Análisis de datos

## Análisis realizados

Durante el proyecto se realizaron los siguientes análisis:

### 1. Ventas por marca
Se agruparon los datos por marca para identificar cuáles tuvieron mayor volumen de ventas.

### 2. Ventas por ciudad
Se analizaron las ventas por ciudad para entender el comportamiento geográfico.

### 3. Ventas por año
Se evaluó cómo han cambiado las ventas a lo largo del tiempo.

### 4. Promedio de especificaciones
Se calcularon promedios de características como:
- RAM
- almacenamiento
- batería

## Objetivo del análisis
El objetivo fue extraer información útil a partir de los datos procesados en AWS.
