# Conclusiones

- AWS permitió trabajar con un entorno real en la nube.
- EC2 facilitó la ejecución del procesamiento de datos.
- S3 permitió almacenar y organizar la información.
- Python y PySpark fueron herramientas clave para el análisis.
- PostgreSQL permitió estructurar los datos para futuras consultas.
- El proyecto permitió aplicar conceptos de Big Data de forma práctica.
